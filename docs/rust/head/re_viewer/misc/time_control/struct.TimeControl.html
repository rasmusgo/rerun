Struct re_viewer::misc::time_control::TimeControl

source · 
pub struct TimeControl {
    timeline: Timeline,
    states: BTreeMap<Timeline, TimeState>,
    playing: bool,
    following: bool,
    speed: f32,
    looping: Looping,
}
Expand description

Controls the global view and progress of the time.

Fields§

§timeline: Timeline

Name of the timeline (e.g. “log_time”).

§states: BTreeMap<Timeline, TimeState>§playing: bool

If true, we are either in PlayState::Playing or PlayState::Following.

§following: bool

If true, we are in “follow” mode (see PlayState::Following). Ignored when [Self.playing] is false.

§speed: f32§looping: Looping
